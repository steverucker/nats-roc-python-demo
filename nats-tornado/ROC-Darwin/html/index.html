ROC SDK Documentation

Introduction

Welcome to the ROC SDK documentation! There are five primary ways to use our software:

  • Application Programming Interface - All ROC SDK capabilities are accessible via a C-language Application Programming Interface (API). This approach is well suited for software engineers looking to build custom applications using ROC SDK face recognition functions.
  • API Wrappers - Experimental Java, C# and Python interfaces that wrap our C API.
  • Command Line Interface - A robust Command Line Interface (CLI) wrapping key ROC SDK API functions. The CLI is ideal for headless server environments or scripts that perform face recognition tasks.
  • Graphical User Interface - An experimental Graphical User Interface (GUI) is shipped with ROC SDK that allows a user to easily ingest/enroll image directories, and perform face recognition applications. This interface is relevant to users seeking an out-of-the-box demonstration of the ROC algorithms.
  • Web Interface - An HTTP interface supporting one-to-one comparison for users wishing to reproduce our online demo locally.
