roc-represent-video

Construct templates from videos.

This command line application corresponds to the roc_represent function in the Application Programming Interface. For representing images from the command line see roc-represent.

Options

Option Meaning Default
-a Use ROC_FULL face detector Use frontal face detector
-p Use ROC_PARTIAL face detector Don't use partial face detector
-f <float> Specify face detection False Detection Rate 0.02
-i Use ROC_ID face representation Use ROC_FR face representation
-k <int> Specify the Maximum Faces per frame 1
-m <int> Specify face detection Minimum Size in pixels 36
-n <float> Specify face detection Adaptive Minimum Size 0.04
-o Estimate pitch and yaw with ROC_PITCHYAW Don't estimate pitch and yaw
-s <int> Process video frames one in every Time Step 0
-l <int> Specify the Capture Length of video to capture in milliseconds. INT_MAX
-t Construct face thumbnail with ROC_THUMBNAIL Don't construct face thumbnail
-c Estimate occlusion with ROC_OCCLUSION Don't estimate occlusion

Time Step

When ingesting videos, roc-represent-video will only process one frame in every s milliseconds.

  • If s is 0, only key frames will be processed.
  • If s is 1, every frame will be processed.

Capture Length

When ingesting videos streamed from a webcam or IP camera, l should be set to the desired length of video to capture in milliseconds.

Examples

Show help
Print application usage and supported arguments. 
$ roc-represent-video -h
Represent a single video file
Represent a single video faces.mp4 to a gallery faces.t. 
$ roc-represent-video faces.mp4 faces.t
Represent multiple files
Represent a folder of videos videos/* to a gallery faces.t. 
$ roc-represent-video videos/* faces.t
Represent from stdin
Represent videos from stdin to a gallery faces.t. 
$ ls -1 videos/* | roc-represent-video faces.t
Represent to stdout
Represent videos from stdin to stdout. 
$ ls -1 videos/* | roc-represent-video > faces.t
Represent multiple faces per video frame
Represent up to 5 faces for each frame in faces.mp4 to a gallery faces.t. 
$ roc-represent-video -k 5 faces.mp4 faces.t
Skip frames in a video
Represent one frame every second in faces.mp4 to a gallery faces.t. 
$ roc-represent-video -s 1000 faces.mp4 faces.t
Represent from an IP camera
Represent frames grabbed from an IP camera for 60 seconds to a gallery faces.t 
* $ roc-represent-video -l 60000 http://204.248.124.202/mjpg/video.mjpg faces.t
