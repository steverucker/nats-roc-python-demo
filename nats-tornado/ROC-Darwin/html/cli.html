Command Line Interface

A robust Command Line Interface (CLI) wrapping key ROC SDK API functions. The CLI is ideal for headless server environments or scripts that perform face recognition tasks.

Workflow

Below is a diagram illustrating the common workflow through our ten command line applications. As you'll soon see, each application is very simple, doing one thing only and (we hope) one thing well. Click on an application to pull up its documentation and example usage.

Applications

Here's a list of the applications along with a brief description of each.

Application Description
roc-at Extract templates in a gallery starting at a specified index.
roc-cluster Cluster a gallery into persons.
roc-consolidate Remove redundant templates for each person in a gallery.
roc-metadata Extract metadata from a gallery.
roc-remove Remove templates in a gallery at the specified indicies.
roc-represent Construct templates from images.
roc-represent-video Construct templates from videos.
roc-search Retrieve a sorted list of the top candidate templates for a probe.
roc-track Track persons in a video gallery.
roc-validate Validate that the contents of a gallery for data corruption.
roc-compare Measure the similarity between galleries.

Gallery Files

A contiguous array of templates written to disk. This file format is shared amongst all command line applications and is identified by convention using a .t suffix.

Concatenation

An important aspect of gallery files is that two or more of them can be concatenated together to produce one larger gallery file. For example: 

$ cat a.t b.t > ab.t

Interoperability

Gallery files produced from the Command Line Interface are interoperable with those produced from the Application Programming Interface. See roc_gallery_file for details.
