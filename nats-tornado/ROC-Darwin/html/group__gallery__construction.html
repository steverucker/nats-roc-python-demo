Detailed Description

Open, read, edit and close galleries.

Open a roc_gallery_file with roc_open_gallery or roc_temporary_gallery, and close it with roc_close_gallery. Treat a gallery as a list of templates with roc_enroll, roc_size and roc_at, and roc_remove. Query the template limit with roc_template_limit.

Typedef Documentation

typedef const char* roc_gallery_file

A contiguous array of templates written to disk.

Iterate through templates in a gallery file using roc_read_template and roc_write_template. Open gallery files for analysis using roc_open_gallery.

typedef struct roc_gallery_type* roc_gallery

A list templates.

Templates are held in memory for efficient search.

Initialize with roc_open_gallery, search with roc_search, and free with roc_close_gallery.

An immutable gallery is a roc_const_gallery.

typedef void(* roc_progress) (float)

Pointer to a callback function used to report progress on a long running task.

The function should return void and take one float parameter which will range from 0.0 to 100.0 indicating the percent completed.

Any function taking a roc_progress parameter will accept a NULL value indicating not to report progress.

Example
void print_progress(float percent_completed)
{
printf("%.1f percent complete!\n", percent_completed);
}
...
roc_gallery_file gallery_file = ...;
roc_gallery gallery;
roc_open_gallery(gallery_file, &gallery, &print_progress);
// `roc_open_gallery` will periodically call `print_progress` indicating
// the percent completed.
typedef size_t roc_template_index

The index of a template in a gallery.

Retrieve the corresponding template with roc_at. The value ROC_INVALID_TEMPLATE_INDEX indicates an invalid template index.

roc_error roc_enroll ( roc_gallery  gallery,
const roc_template  template_ 
)

Add a template to a gallery.

This function is similar to roc_write_template, appending the template to the end of the corresponding roc_gallery_file. The enrolled template is now a potential candidate returned by roc_search.

On platforms that support it, this function calls fsync on the gallery file after writing template_. Even if the application closes without calling roc_close_gallery, the biometric data will be preserved.

Templates are stored in gallery akin to a std::vector. The roc_template_index of template_ in gallery will be equal the roc_size of gallery just prior to calling this function.

To remove an enrolled template see roc_remove.

Example
roc_gallery gallery = ...;
roc_template template_ = ...;
roc_template_index template_index;
roc_size(gallery, &template_index);
roc_enroll(gallery, template_);
Parameters
[in]galleryGallery to add template_ to.
[in]template_Template to add to gallery.
Remarks
This function is reentrant.

Retrieve a template from a gallery at the specified index.

This function returns a copy of the gallery template. Free the copy after use with roc_free_template.

A gallery preserves the order of templates in its corresponding roc_gallery_file.

This function is exposed in the Command Line Interface as roc-at.

Example
roc_gallery gallery = ...;
roc_template_index index = ...;
roc_template template_;
roc_at(gallery, index, &template_);
Parameters
[in]galleryThe gallery to retrieve a template from.
[in]indexThe index of the template, between 0 and (roc_size - 1).
[out]template_A copy of the template in gallery at index.
Remarks
This function is thread-safe.

Remove the template at the specified index.

For efficiency purposes, this function will replace the removed template with an Empty Template. Therefore roc_size will remain the same, as will the roc_template_index of the remaining templates.

This function is exposed in the Command Line Interface as roc-remove.

roc_error roc_template_limit ( size_t *  template_limit)

Query the remaining template limit.

Most licenses of the ROC SDK include a limit on the number of templates that can be used at once. The number of templates in use is the sum of roc_size for all galleries in use. This function returns the number of templates specified in the license file less the number of templates in use.

Example
size_t template_limit;
roc_template_limit(&template_limit);
Parameters
[out]template_limitThe remaining number of templates available.
Remarks
This function is thread-safe.
