API Wrappers

Experimental Java, C# and Python interfaces that wrap our C API.

Overview

The ROC SDK ships with API wrappers for the Java, C# and Python programming languages. The wrappers live in the java, csharp and python subdirectories of the SDK respectively. In addition to the language-specific interface files, each folder also contains an examples subfolder with translations of examples/roc_example_verify.c and examples/roc_example_search.c. Lastly, source code for the API translation is provided in the roc_swig.c file as a reference for resolving specific technical questions.

At this time, the Video Decoding portion of the C API is not supported in the wrappers.

Reproducibility

The wrappers are generated automatically using SWIG, thus it is trivial to reproduce them or generate new wrappers for other programming languages. The general syntax is:

$ cd include
$ swig -<language> -module roc roc.h

Help

To the extent possible, symbol names are identical between the wrappers and the C API, thus the Application Programming Interface documentation should continue to serve as the primary reference. A reading of SWIG Basics and the language specific documentation for Java, C# and Python is encouraged.

The most difficult component of these wrappers is understanding how SWIG handles pointers and arrays. Cross-referencing the following documents will offer clarity:

  • The SWIG documentation for %pointer_functions and %array_functions.
  • The use of %pointer_functions and %array_functions in roc.h, also reproduced here.
  • The provided verify and search examples that make use of these functions.

If in doubt, please direct questions to suppo.nosp@m.rt@r.nosp@m.ankon.nosp@m.e.io.

Examples

Java

$ cd java
$ javac *.java examples/roc_example_*.java
$ java roc_example_verify ../data/josh_1.jpg ../data/josh_2.jpg
$ java roc_example_search ../data/roc.jpg ../data/josh_2.jpg

C#

Using the Mono compiler.

$ cd csharp
$ mcs -platform:x64 *.cs examples/roc_example_verify.cs -out:roc_example_verify.exe
$ mcs -platform:x64 *.cs examples/roc_example_search.cs -out:roc_example_search.exe
$ mono roc_example_verify.exe ../data/josh_1.jpg ../data/josh_2.jpg
$ mono roc_example_search.exe ../data/roc.jpg ../data/josh_2.jpg

When calling mono on OS X include the --arch=64 flag.

Python

On Windows be sure to use a 64-bit installation of Python.

$ cd python
$ python examples/roc_example_verify.py ../data/josh_1.jpg ../data/josh_2.jpg
$ python examples/roc_example_search.py ../data/roc.jpg ../data/josh_2.jpg
