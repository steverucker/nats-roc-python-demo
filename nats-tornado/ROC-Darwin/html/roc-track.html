roc-track

Track persons in a video gallery.

This command line application corresponds to the roc_track function in the Application Programming Interface.

Options

Option Meaning Default
-t <int> Specify Maximum Time Separation between contiguous faces in a track 1000
-o <int> Specify Minimum Detection Overlap between contiguous faces in a track 0.5
-s <int> Specify Minimum Similarity between contiguous faces in a track 0.5

Examples

Show help
Print application usage and supported arguments. 
$ roc-track -h
Track a gallery
Track a gallery video.t and write its persons to persons.txt. 
$ roc-track faces.t persons.txt
Track a gallery to stdout
Track a gallery video.t and print its persons to stdout. 
$ roc-track faces.t
