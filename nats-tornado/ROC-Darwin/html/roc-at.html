roc-at

Extract templates in a gallery starting at a specified index.

This command line application corresponds to the roc_at function in the Application Programming Interface.

Options

Option Meaning Default
-f Extract feature vector Extract template
-k <int> Specify the Number of Templates to extract 1
-s <int> Specify the Template Step to skip templates 1
-t Extract thumbnail Extract template

Number of Templates

Retrieve k templates starting at the specified index. If k is -1 all templates starting at the specified index will be retrieved. If k extends past the end of the gallery then less than k templates will be retrieved.

Template Step

Retrieve one in every s templates starting at the specified index.

Examples

Show help
Print application usage and supported arguments. 
$ roc-at -h
Extract a template
Write the fourth template in the gallery faces.t to fourth.t. 
$ roc-at faces.t 3 fourth.t
Extract multiple templates
Write the first two templates in the gallery faces.t to first_two.t. 
$ roc-at -k 2 faces.t 0 first_two.t
Extract every other template
Extract every other template in the gallery faces.t to every_other.t. 
$ roc-at -k -1 -s 2 faces.t 0 every_other.t
Extract a template to stdout
Print the fourth template in the gallery faces.t to stdout. 
$ roc-at faces.t 3 > fourth.t
